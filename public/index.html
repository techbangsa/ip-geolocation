API Reference

POST
/api/projects

GET
/api/location
Auto-detect visitor IP. Header: x-api-key: YOUR_KEY
GET
/api/location?ip=8.8.8.8
Look up specific IP. Header: x-api-key: YOUR_KEY
GET
/api/projects/stats
Your usage stats. Header: x-api-key: YOUR_KEY
GET
/health
Server health check. No auth needed.
